#How to run the app

1. set SPOTIFY_CLIENT_ID and SPOTIFY_CLIENT_SECRET in your local environment.
2. run 'npm install' command in both the folder (auth_server & spotify_leagueX).
3. go to 'auth_server' directory and run 'npm start' command.
4. go to 'spotify_leagueX' directory and run 'npm run start' command.
5. enter 'http://localhost:8000' on your browser, login to Spotify.
    Then it will redirect you to client web app.
